목록관련태그

개요

깔끔한 정보전달을 위해 여러 요소들을 일렬로 나열한 명단이나 목록을 의미

html에서는 목록을 표현하기 위해 순서 없는 목록, 순서 있는 목록, 정의목록 등을 지원

순서 없는unordered 목록

정의definition목록

용어와 그 용어에 대한 해설을 모아놓은 목록

dl태그로 목록의 시작, dt태그로 용어의 이름, dd태그로 해당 용어에 대한 설명을 작성

웹표준 용어

HTML
웹사이트를 구성하는 웹문서 작성에 사용하는 마크업 언어
CSS
html로 작성된 문서를 시각적으로 꾸밀 때 사용
Javascript
html에서 사용하는 객체기반 스크립트 프로그래밍 언어
