테이블관련태그

개요

여러 종류의 데이터를 일목요연하게 2차원 형태의 표로 나타내기 위해 사용

table태그로 테이블 시작, th태그로 첫열(제목), tr태그로 행, td태그로 본문(셀)을 나타냄

table의 디자인 관련 속성은 CSS를 이용(width, height, padding, border-collapse...)

복잡한 표 만들기

복잡한 형태의 테이블을 작성하려면 colspan/rowspan속성을 적절히 사용

colspan은 여러 열을 합쳐 하나의 열로 만드는 것

rowspan은 여러 행을 합쳐 하나의 열로 만드는 것

테이블 너비 지정

테이블 요소의 크기를 지정하는 단위는 px, %임

px로 지정하는 경우, 고정크기로 설정되어 브라우저 크기에 상관없이 테이블의 크기도 변화없음

%로 지정하는 경우, 가변크기로 설정되어 브라우저 크기에 따라 테이블의 크기도 변함
